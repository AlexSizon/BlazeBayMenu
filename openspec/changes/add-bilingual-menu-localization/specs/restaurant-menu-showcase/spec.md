## MODIFIED Requirements

### Requirement: Branded web menu presentation
The system SHALL present Blaze Bay as a branded web menu experience that reflects the printed menu's dark neon beach-bar identity while rendering its hero copy and supporting interface text in the active language.

#### Scenario: Guest opens the menu for the first time
- **WHEN** a guest visits the web menu on a phone or desktop browser
- **THEN** the page shows Blaze Bay branding, localized menu context, and a high-contrast neon-on-dark visual treatment inspired by the printed menu
